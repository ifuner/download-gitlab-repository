# 下载公司内网gitlab所有仓库

> 使用前请先配置 ./config.js 文件

```bash
 npm install
 
  # 单线程下载
 npm run download
 
 # 多线程下载
 npm run download:wokers
```

# Download all warehouses of the company's intranet gitlab

> Please configure the ./config.js file before use

```bash
 npm install
 
# Single thread download
 npm run download
 
 # Multi-threaded download
 npm run download:wokers
```
